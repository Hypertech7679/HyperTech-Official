About Nmap

Nmap (Network Mapper) is a powerful open-source tool used for network discovery and security auditing. It is widely used by security professionals and system administrators to map networks and discover open ports, services, and vulnerabilities. Below is a step-by-step guide to help you set up Nmap and run a basic scan on your network.

Step-by-Step Guide: Nmap Setup & Basic Scanning

  1. Install Nmap:
    • For Linux: Open a terminal and run the command: sudo apt install nmap.
    • For Windows: Download the Windows installer from the official Nmap site here.
    • For macOS: Use Homebrew to install Nmap by running: brew install nmap.
  2. Verify Installation:

    Open a terminal (or Command Prompt on Windows) and type nmap --version. If Nmap is correctly installed, it will display the version and license information.

  3. Basic Network Scan:

    To scan your local network or a specific IP address, run the following command in the terminal:

    nmap 192.168.1.1

    Replace 192.168.1.1 with the IP address of the target you want to scan.

  4. Scanning for Open Ports:

    To detect open ports on a target, use this command:

    nmap -p 1-65535 192.168.1.1

    This will scan all 65535 TCP ports on the target system. You can limit the scan to common ports by omitting the range.

  5. Scan Multiple Targets:

    To scan multiple IP addresses or ranges, use the following command:

    nmap 192.168.1.1 192.168.1.2 192.168.1.0/24

    This will scan the two specified IP addresses and the entire 192.168.1.0/24 subnet.

  6. Advanced Scan (Service Detection):

    If you want to identify running services on open ports, use:

    nmap -sV 192.168.1.1

    This will probe each open port and attempt to identify the running service and version.

  7. Save Scan Results:

    To save scan results to a file, use the -oN option:

    nmap -oN scan_results.txt 192.168.1.1

    This will save the output to a text file named scan_results.txt.
